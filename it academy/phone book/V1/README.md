# V1

### How to run
1. Create `venv`
    ```bash
    python -m venv env
    ```
1. Install dependensies
    ```bash
    pip install -r requirements.txt
    ```
1. Activate `venv`
1. Run
    ```bash
    python phone_book.py
    ```
